# Requisitos Não Funcionais (RNF)

## RNF01 - Desempenho
O sistema deve gerar respostas da Inteligência Artificial em tempo adequado para não comprometer a experiência do usuário.

## RNF02 - Usabilidade
O aplicativo deve possuir interface simples e intuitiva para facilitar o uso pela equipe de marketing.

## RNF03 - Disponibilidade
O sistema deve estar disponível para uso da equipe da agência sempre que necessário.

## RNF04 - Segurança
O sistema deve garantir autenticação segura para acesso dos usuários.

## RNF05 - Escalabilidade
O sistema deve permitir crescimento do número de usuários e geração de conteúdos sem perda significativa de desempenho.

## RNF06 - Armazenamento de Dados
O sistema deve armazenar o histórico de conteúdos gerados de forma segura.

## RNF07 - Compatibilidade
O sistema deve funcionar em dispositivos móveis compatíveis com as principais versões de sistemas operacionais mobile.
