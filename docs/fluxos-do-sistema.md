# Fluxos do Sistema

## Fluxo de Cadastro

1. Usuário abre o aplicativo
2. Usuário escolhe criar conta
3. Usuário preenche dados
4. Sistema valida informações
5. Conta é criada

## Fluxo de Publicação de Anúncio

1. Usuário acessa área de anúncios
2. Usuário cria novo anúncio
3. Usuário adiciona imagem e descrição
4. Sistema publica o anúncio
